Демонстрація Системи Шифрування Електронної Пошти

Опис
Цей інструмент реалізує систему захисту електронного листування для демонстрації принципів симетричного шифрування та конфіденційності даних.
Програма виконує такі функції:
- Генерація унікального криптографічного ключа на основі персональних даних (Email) з використанням хешування SHA-256.
- Шифрування текстових повідомлень за допомогою симетричного алгоритму Fernet (AES).
- Розшифрування отриманих повідомлень та відновлення початкового тексту.
- Демонстрація процесу безпечного обміну даними та перевірка цілісності повідомлення.

Системні Вимоги
- Python 3.8+
- Бібліотека: cryptography (встановлюється через `pip install cryptography`)

Запуск Програми
bash
python main.py

Інструкція з Використання
Після запуску програма працює в інтерактивному режимі та виконує наступні кроки:

1. [Крок 1] Користувач вводить свою електронну пошту (або використовує дефолтну). На її основі генерується унікальний ключ шифрування.
2. [Крок 2] Користувач вводить текст повідомлення, яке необхідно захистити.
3. [Крок 3] Програма шифрує повідомлення, перетворюючи його на нечитабельний набір символів (ciphertext).
4. [Крок 4] Програма емулює отримання листа адресатом та розшифровує його, використовуючи згенерований ключ.
5. Здійснюється автоматична перевірка: чи співпадає розшифрований текст із оригіналом.

Система Оцінювання
Методи симетричного шифрування (0–20 балів)

Генерація ключів (0–10 балів)
- Використання хешування (SHA-256) від персональних даних гарантує, що ключ є унікальним для кожного користувача, але відновлюваним при правильному введенні даних.

Конфіденційність та цілісність (0–10 балів)
- Алгоритм Fernet забезпечує високий рівень захисту: без ключа прочитати повідомлення неможливо.
- Програма демонструє, що розшифровані дані повністю відповідають відправленим, що підтверджує цілісність інформації.

Висновок
У ході лабораторної роботи я дослідив принципи захисту електронної кореспонденції та реалізував програмний засіб для шифрування повідомлень мовою Python.
Було використано бібліотеку `cryptography` для створення симетричного шифратора, який генерує ключі на основі електронної адреси користувача.
Робота продемонструвала, як перетворення відкритого тексту на шифротекст захищає конфіденційність листування від сторонніх осіб.
