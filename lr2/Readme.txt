Аналізатор Захищеності Повідомлень
Програма для дослідження класичних методів симетричного шифрування та оцінки їх криптостійкості.

Опис
Цей інструмент виконує порівняльний аналіз двох алгоритмів шифрування:
Шифр Цезаря – простий метод заміни символів із фіксованим зсувом.
Шифр Афіна – математичний метод, що використовує множення та додавання з модульною арифметикою.

Програма дозволяє згенерувати ключі на основі персональних даних користувача (імені та прізвища), виконати шифрування і розшифрування тексту, а також провести частотний аналіз для оцінки ефективності шифру.

Системні Вимоги
Python 3.8+
Стандартні бібліотеки: string, collections

Запуск Програми
bash
python cipher.py

Інструкція з Використання
Після запуску програма послідовно виконує такі дії:

Зчитує ім’я та прізвище користувача.

Генерує ключі для шифрів Цезаря та Афіна.

Шифрує заданий текст та виводить результат.

Розшифровує текст для перевірки коректності.

Аналізує частотний розподіл символів та формує таблицю порівняння.

Система Оцінювання
Методи шифрування (0–20 балів)
Шифр Цезаря (0–10 балів)
Простий у реалізації, але легко зламується перебором ключів. Має низьку криптостійкість через обмежену кількість можливих варіантів.
Шифр Афіна (0–10 балів)
Використовує два параметри (a і b), що створює більше комбінацій ключів і підвищує рівень безпеки. Проте також вразливий до частотного аналізу.
Загальна оцінка визначається за результатами аналізу ефективності шифрування та правильності розшифрування.
Висновок
У ході лабораторної роботи я навчився створювати програми для симетричного шифрування повідомлень, генерувати ключі на основі персональних даних та проводити частотний аналіз зашифрованого тексту. Було проведено порівняння шифрів Цезаря та Афіна, визначено їх переваги й недоліки, а також зроблено висновок, що обидва методи підходять лише для базового захисту інформації, оскільки не забезпечують високої криптостійкості.
